# My Favorite Eats ##

This app will allow you simulate a Bitcoin mining operation just for fun.


## Installation ##

```shell
git clone https://github.com/Donnadieu/rails-miner-game.git
cd rails-miner-game
bundle install
rails db:migrate
rails s
```
